# About This Field Guide

## What is the Resilience Biomarkers Field Guide?

The Resilience Biomarkers Field Guide is a comprehensive resource for discovering and validating biomarkers of resilience in aquaculture species using omics data. This guide bridges the gap between theoretical concepts and practical application, providing researchers with actionable workflows and methodologies.

## Scope

This field guide covers:

### Data Sources
- Public omics databases and repositories
- Experimental design considerations for stress studies
- Data quality assessment and preprocessing

### Analytical Approaches
- Transcriptomics analysis workflows
- Proteomics data integration
- Metabolomics profiling
- Multi-omics integration strategies

### Biomarker Discovery
- Statistical approaches for identifying candidate biomarkers
- Validation strategies
- Meta-analysis methods for cross-study integration
- Prioritization frameworks

### Application
- Biomarker-assisted selection in breeding programs
- Environmental monitoring and early warning systems
- Resilience assessment tools

## Development and Maintenance

This field guide is:

- **Open and collaborative**: Contributions and feedback are welcome
- **Living documentation**: Regularly updated with new methods and findings
- **Evidence-based**: Grounded in published research and validated approaches
- **Reproducible**: Includes code examples and detailed protocols where applicable

## How to Use This Guide

The field guide is organized into chapters covering different aspects of biomarker discovery and application. Each chapter includes:

- Background and context
- Step-by-step methodologies
- Best practices and common pitfalls
- Example datasets and code (where applicable)
- References to primary literature

You can navigate through the guide using the navigation menu, or use the search functionality to find specific topics.
