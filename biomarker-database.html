Biomarker Database

This database compiles candidate resilience biomarkers discovered across shellfish genomic studies. Biomarkers are searchable by species, condition, trait, discovery class, and evidence type. Use the search bar and column sort features to explore entries, and download the full or filtered dataset using the buttons below. To suggest additions or provide feedback, use the Feedback / Request Update link.

How to use this database
  • Search: Use the search box (top right) to query any text across all columns.
  • Sort: Click any column header to sort the table by that column.
  • Filter: Use the dropdown filters above each column to narrow results by species, condition, class, etc.
  • Download: Use the CSV, Excel, or Copy buttons to download the current (filtered) view.
  • Feedback: Use the Feedback / Request Update link to suggest a new biomarker entry or report an issue.

Rank — confidence score (1–5) based on the number of supporting studies, species, and data types; higher scores indicate stronger evidence. Source — publication(s) providing original evidence.

Database hosted on GitHub. Stable annual releases are made to ensure data, webpage, and software are up to date. To contribute data or join as a collaborator, open a GitHub Issue.
